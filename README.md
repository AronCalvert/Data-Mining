# CS4168 Project

## Data set

We are using the dataset tracks2026.csv available on Brightspace. The dataset
contains audio features and metadata for music tracks across multiple genres.
This dataset is an excerpt of the Spotify Tracks Dataset available at:
https://huggingface.co/datasets/maharshipandya/spotify-tracks-dataset.  The link to Hugging Face is included here only for reference.
